                   HTTP Message Signatures Directory
     draft-meunier-httpbis-http-message-signatures-directory-latest

Abstract

   This document describe a method for clients using using
   [HTTP-MESSAGE-SIGNATURE] to advertise their signing keys.  It defines
   a key directory format based on JWKS, as well as new HTTP Method
   Context to allow for in-band key discovery.
